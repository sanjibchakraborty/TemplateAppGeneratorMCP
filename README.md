# TemplateAppGeneratorMCP

MCP server for the iOS generator wizard in Cursor.

```bash
git clone <repo-url>
cd TemplateAppGeneratorMCP/mcp
npm install && npm run build
```

Add **`ios-app-generator`** to **`~/.cursor/mcp.json`** (`url` + `env`), then run **`npm run start:http`**. Details: **[mcp/README.md](./mcp/README.md)**.
